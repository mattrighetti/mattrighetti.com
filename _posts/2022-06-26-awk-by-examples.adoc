Let's start right off

=== Print all lines
```awk code
awk '{ print }' data.csv
```
Which is the same of
```awk code
awk '{ print $0 }' data.csv
```
This will print out every line, in `awk` the value `$0` stores the entire line.

=== Count number of processes
```bash
$ cat data.txt | wc -l
```
You can get the same result with
```awk code
END { print NR }
```


=== Select fields
```awk code
{ print NR, $0 }
```
